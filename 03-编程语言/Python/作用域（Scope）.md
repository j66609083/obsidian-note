> A scope is a textual region of a Python program where a namespace is directly accessible


解释器顶层调用的语句，无论是从脚本文件读取还是交互式执行，都被视为一个名为 `__main__` 的模块的一部分，因此它们拥有自己的全局命名空间。（内置名称实际上也存在于一个模块中；这被称为 `builtins` 。）

函数的局部命名空间在函数被调用时创建，并在函数返回或抛出未被函数内部处理的异常时被删除
