
### pass语句

`pass` 语句什么也不做。当语法上需要一个语句但程序不需要任何操作时，可以使用它。例如：
```python
>>> while True:
...    pass  # Busy-wait for keyboard interrupt (Ctrl+C)
...
```


这通常用于创建最小的类：
```python
class MyEmptyClass:
    pass
```


`pass` 的另一个用途是作为新代码中函数或条件体的占位符，让你能保持更抽象的思考层次
```python
def initlog(*args):
    pass   # Remember to implement this!
```


### match语句
一个 `match` 语句接受一个表达式，并将其值与一个或多个作为 case 块提供的模式进行比较
只有第一个匹配的模式会被执行，并且它还可以将值中的组件（序列元素或对象属性）提取到变量中
如果没有 case 匹配，则没有任何分支被执行。

```python
def http_error(status):
    match status:
        case 400:
            return "Bad request"
        case 404:
            return "Not found"
        case 418:
            return "I'm a teapot"
        case _:
            return "Something's wrong with the internet"
```

注意最后一个块：变量名 `_` 充当通配符，并且永远不会失败匹配

你可以使用 `|` （“或”）将多个字面量组合在单个模式中
```python
case 401 | 403 | 404:
    return "Not allowed"
```


默认值在定义作用域中的函数定义点被计算，因此以下代码将打印5
```python
i = 5

def f(arg=i):
    print(arg)

i = 6
f()
```
