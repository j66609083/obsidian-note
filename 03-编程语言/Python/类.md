
类永远不会被用作全局作用域

方法和普通函数一样，可以引用全局名称。与一个方法关联的全局作用域是包含其定义的模块

每个值都是一个对象，因此它有一个类（也称为它的类型）。它被存储为 `object.__class__`



## 继承

有两个与继承相关的内置函数

使用 `isinstance()` 来检查实例的类型： `isinstance(obj, int)` 将会是 `True` 仅当 `obj.__class__`是 `int` 或从 `int` 派生的某个类时。
使用 `issubclass()` 来检查类继承： `issubclass(bool, int)` 是 `True` ，因为 `bool` 是 `int` 的子类。但是， `issubclass(float, int)` 不是 `False` ，因为 `float` 不是 `int` 的子类。




支持多重继承：
```python
class DerivedClassName(Base1, Base2, Base3):
    <statement-1>
    .
    .
    .
    <statement-N>
```
