
文件名以`.rs`结尾
rust是一个提前编译的语言（ahead-of-time compiled）


### 变量
1. 默认是不可变的（给不可变变量赋值时编译会报错）
2. 可以通过在变量名前加上`mut`变成可变变量


### 常量
1. 使用`const`关键字
2. 值的类型必须被注解
3. 可以在任何作用域中声明
4. 常量只能设置为常量表达式，而不能设置为只能在运行时计算出的值的结果
5. 命名约定是使用全大写字母，单词之间用下划线分隔
6. 有助于解释硬编码的值的意义
```rust
const THREE_HOURS_IN_SECONDS: u32 = 60 * 60 * 3;
```


### 遮蔽（Shadowing）
1. 可以用与之前变量相同的名字声明一个新变量
2. 与`mut`不同，重新赋值不会编译报错
3. 实际上是在创建一个新变量，因此可以改变值的类型但重用相同的名称


### 数据类型（Data Types）
1. 分为2种：标量（scalar）和复合类型（compound）

#### 标量类型
1. 表示单个值
2. 有4种主要的标量类型：整数，浮点数，布尔值和字符
